IDEA 配置Tomcat

IDEA 配置Tomcat

  1. 选择菜单栏选项Run -> Edit Configuration

  2. 点击左侧“+”，选择Tomcat Server -> Local

  3. 在Tomcat Server -> Unnamed -> Server -> Application server项目下，点击 Configuration 按钮

    可以改名为 LocalTomcat

  4. 选择本地Tomcat服务器,确定

  5. 至此,IntelliJ IDEA配置Tomcat完成
